## Under the hood

`tailspin` uses:

* [`less`](http://greenwoodsoftware.com/less/) for viewing
* [Bubble Tea](https://github.com/charmbracelet/bubbletea) for handling terminal control over to `less`
* [cobra](https://github.com/spf13/cobra) for the CLI
